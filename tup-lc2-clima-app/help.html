Ayuda Clima App V.1.0

Esta App te permite consultar el clima de una cuidad en particular utilizando la API de OpenWeatherMap.

Para poder consultar el clima, primero deberá Agregar un cuidad y lugeo seleccionar la Cuidad en cuestión desde la página Clima
